# 样例班表说明

`2026年班表.xlsx` 为真实格式样例，用于对照解析逻辑。

## 表头结构（以「张小燕」行为例）

- 第 2 行：日期 `1`～`31`，其后依次为 **应出勤 / 实际出勤 / 本月余 / 累计余**（列位置随当月天数变化，如 2 月汇总从第 31 列起）。
- 员工行：左侧为每日班次（值/白/休等），右侧汇总格可能为：
  - 数字：`180`、`152`、`0`
  - 文本：**余4**、**余22**、**余26**（表示结余工时）
  - 文本：**负4**（表示欠工时）

## 期望解析结果（张小燕）

| Sheet | 应出勤 | 实际 | 本月余 | 累计余 |
|-------|--------|------|--------|--------|
| 1月   | 180    | 184  | +4     | +22    |
| 2月   | 152    | 148  | -4     | +18    |
| 5月   | 152    | 152  | 0      | +26    |

顶部「累计余」应取 **5 月** 的 `余26`（最新有数据的月份）。

## 星期行

- 星期在日期行**上一行**（单字：一、二、三…日）
- 解析时按「整格是星期字」匹配，避免把日期数字误当成星期
